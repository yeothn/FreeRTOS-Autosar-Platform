# Project References

This project is implemented based on the following AUTOSAR Classic Platform standards.

## 1. Operating System (OS) Specification
* **Document Title:** Specification of Operating System (SWS_Os)
* **Target Version:** **R24-11** (Release 2024, November)
* **File Name:** `AUTOSAR_SWS_OS.pdf`
* **Download Source:** [AUTOSAR Official Standard R24-11](https://www.autosar.org/fileadmin/standards/R24-11/CP/AUTOSAR_CP_SWS_OS.pdf)

## 2. Runtime Environment (RTE) Specification
* **Document Title:** Specification of RTE Software (SWS_RTE)
* **Target Version:** **R24-11** (Release 2022, November)
* **File Name:** `AUTOSAR_SWS_RTE.pdf`
* **Download Source:** [AUTOSAR Official Standard R24-11](https://www.autosar.org/fileadmin/standards/R24-11/CP/AUTOSAR_CP_SWS_RTE.pdf)

## 3. Methodology & Compliance
* **OSEK/VDX Compatibility:** This implementation also references the **OSEK/VDX OS 2.2.3** specification, as AUTOSAR OS is backward compatible with OSEK.
