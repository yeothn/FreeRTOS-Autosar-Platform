# Project References

This project is implemented based on the following AUTOSAR Classic Platform standards.

## 1. Operating System (OS) Specification
* **Document Title:** Specification of Operating System (SWS_Os)
* **Target Version:** **R22-11** (Release 2022, November)
* **File Name:** `AUTOSAR_SWS_OS.pdf`
* **Download Source:** [AUTOSAR Official Standard R22-11](https://www.autosar.org/fileadmin/standards/R22-11/CP/AUTOSAR_SWS_OS.pdf)

## 2. Methodology & Compliance
* **OSEK/VDX Compatibility:** This implementation also references the **OSEK/VDX OS 2.2.3** specification, as AUTOSAR OS is backward compatible with OSEK.
